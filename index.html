CAAR - The modern LISP machine

CAAR is an attempt at writing a modern LISP machine from scratch. The architecture's small size and documentation allows for anyone to easily make an assembler or emulator.

Philosophy

The goal of this project is not to make a super fast architecture or have the most efficient design, but rather take small elements and build bigger ones by combining them. For example, the typical mov instruction is really just a macro for push and pop in the CAAR assembler.

Features

  • 32-bit memory addresses
  • Sexp-based assembler
  • MISC architecture (~25 instructions)
  • Hardware-based optimizations for cons cells operations such as cons, car and cdr

Planned features

  • R5RS-compliant Scheme compiler
